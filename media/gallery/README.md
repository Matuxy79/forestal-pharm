# Gallery seed media

Drop real photos or videos of the shop here (`.jpg`, `.png`, `.webp`, `.mp4`, `.mov`, `.webm`) and
they'll automatically show up when the app starts, and in "Show Gallery" — no code changes needed.

The current `.jpg` files in this folder were harvested from the pharmacy's real Facebook page as
seed content. Swap in your dad's actual shop videos whenever you're ready — just add or replace
files here.
